Audit & Edit Guide

A complete reference of every piece of text on the website — organized by page, section location, and display format. Use the edit boxes to write your updated copy, then share back with your team to implement.

How To Use This Document

1. Each block shows WHAT the content is, WHERE it sits on the page, and HOW it's displayed to visitors.

2. The gold box contains the current live text exactly as it appears on the site.

3. Type your changes directly into the green "Your Edit" box below each section. Leave blank if no change needed.

4. Use the Notes field for any instructions to your team (e.g. "replace with photo," "confirm this stat," "delete this section").

5. When finished, use the Print / Save as PDF button and send the PDF to your team.

Use your browser's "Save as PDF" option when printing for best results.
